# JS: Functions & Conditionals – Racing Quiz 🏎️

This is a JavaScript project built as part of the Technigo Web Development program. 
It focuses on logic, functions, and user interaction through a playful "racing" theme.

## 🌟 Features
* **The Coders:** Introduction of the driver and navigator.
* **Name Comparison:** Logic that determines who has the longest name.
* **Age Check:** A conditional flow that reacts differently based on the user's age.
* **The Math Engine:** A random math challenge (0-10) that the user must pass to "start the engine".
* **Bonus Logic:** Integrated age verification within the math test to ensure the driver is 18+ to proceed.

## 🛠️ Tech Stack
* **JavaScript:** Logic, math functions, and DOM interaction.
* **HTML5:** Structure of the dashboard.
* **CSS3:** Styled with a modern "Glassmorphism" look using Flexbox for a clean, centered layout.

## 🧠 Key Learnings
During this project, I practiced:
* Writing and invoking functions with parameters.
* Using `Math.random()` and `Math.floor()` to generate dynamic content.
* Managing nested `if/else` statements for complex logic flows.
* Fixing common syntax errors (like the classic script path issues in HTML).
* Structuring a UI with **Flexbox** (`space-evenly`, `column`).
